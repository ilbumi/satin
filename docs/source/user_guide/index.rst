Overview
--------

SATIn is designed to streamline the image annotation process for computer vision projects. This user guide covers:

- Creating and managing annotation projects
- Using the annotation tools effectively
- Organizing work with tasks
- Exporting annotation data
- Keyboard shortcuts for efficiency
